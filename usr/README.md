# Poema O Bicho

Um aplicativo simples em Flutter para exibir o famoso poema "O Bicho" de Manuel Bandeira, que traz uma reflexão profunda sobre a condição humana.

## Funcionalidades
- Exibição limpa e responsiva do poema "O Bicho".
- Design adaptável para telas móveis e desktop.

## Tecnologias
- Flutter
- Dart

## Como executar
1. Clone o repositório.
2. Execute `flutter pub get`.
3. Execute `flutter run`.
